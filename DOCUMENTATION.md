### 2. –¢–µ—Ö–Ω–∏—á–µ—Å–∫–∏–µ –∏–Ω–¥–∏–∫–∞—Ç–æ—Ä—ã

#### RSI (Relative Strength Index)

**–§–∞–π–ª**: `internal/indicators/rsi.go`

**–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ**: –ò–∑–º–µ—Ä—è–µ—Ç —Å–∏–ª—É —Ç—Ä–µ–Ω–¥–∞ –∏ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –ø–µ—Ä–µ–∫—É–ø–ª–µ–Ω–Ω–æ—Å—Ç—å/–ø–µ—Ä–µ–ø—Ä–æ–¥–∞–Ω–Ω–æ—Å—Ç—å.

**–ê–ª–≥–æ—Ä–∏—Ç–º —Ä–∞—Å—á–µ—Ç–∞**:

1. **–í—ã—á–∏—Å–ª–µ–Ω–∏–µ –∏–∑–º–µ–Ω–µ–Ω–∏–π —Ü–µ–Ω—ã**:
   ```go

   gain = max(change, 0)      // –¢–æ–ª—å–∫–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è
   loss = abs(min(change, 0)) // –¢–æ–ª—å–∫–æ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è (–ø–æ –º–æ–¥—É–ª—é)
   ```

2. **–°–≥–ª–∞–∂–µ–Ω–Ω–æ–µ —Å–∫–æ–ª—å–∑—è—â–µ–µ —Å—Ä–µ–¥–Ω–µ–µ**:
   ```go
   // –ü–µ—Ä–≤—ã–µ period —Å–≤–µ—á–µ–π - –ø—Ä–æ—Å—Ç–æ–µ —Å—Ä–µ–¥–Ω–µ–µ

   avgLoss = sum(losses) / period
   
   // –ü–æ—Å–ª–µ–¥—É—é—â–∏–µ - —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–µ —Å–≥–ª–∞–∂–∏–≤–∞–Ω–∏–µ

   ```

3. **–í—ã—á–∏—Å–ª–µ–Ω–∏–µ RSI**:
   ```go

   RSI = 100 - (100 / (1 + RS))
   ```

**–î–∏–∞–ø–∞–∑–æ–Ω –∑–Ω–∞—á–µ–Ω–∏–π**: 0-100

**–°–∏–≥–Ω–∞–ª—ã**:

### Bollinger Bands

**–§–∞–π–ª**: `internal/indicators/bollinger.go`

**–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ**: –û–ø—Ä–µ–¥–µ–ª—è–µ—Ç –≤–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç—å –∏ —É—Ä–æ–≤–Ω–∏ –ø–µ—Ä–µ–∫—É–ø–ª–µ–Ω–Ω–æ—Å—Ç–∏/–ø–µ—Ä–µ–ø—Ä–æ–¥–∞–Ω–Ω–æ—Å—Ç–∏.

### ATR (Average True Range)

**–§–∞–π–ª**: `internal/indicators/atr.go`

**–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ**: –ò–∑–º–µ—Ä—è–µ—Ç –≤–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç—å —Ä—ã–Ω–∫–∞ –¥–ª—è —É—Å—Ç–∞–Ω–æ–≤–∫–∏ —Å—Ç–æ–ø-–ª–æ—Å—Å–æ–≤.

**–ê–ª–≥–æ—Ä–∏—Ç–º —Ä–∞—Å—á–µ—Ç–∞**:

1. **True Range**:
   ```go
   TR = max(
       High - Low,
       abs(High - PreviousClose),
       abs(Low - PreviousClose)
   )
   ```

2. **ATR (—Å–≥–ª–∞–∂–µ–Ω–Ω–æ–µ —Å—Ä–µ–¥–Ω–µ–µ)**:
   ```go
   // –ü–µ—Ä–≤—ã–µ period –∑–Ω–∞—á–µ–Ω–∏–π - –ø—Ä–æ—Å—Ç–æ–µ —Å—Ä–µ–¥–Ω–µ–µ
   ATR = sum(TR) / period
   
   // –ü–æ—Å–ª–µ–¥—É—é—â–∏–µ - —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–µ —Å–≥–ª–∞–∂–∏–≤–∞–Ω–∏–µ
   ATR = (ATR √ó (period-1) + TR) / period
   ```

**–ü–∞—Ä–∞–º–µ—Ç—Ä—ã**:
- `period` - –ø–µ—Ä–∏–æ–¥ —Ä–∞—Å—á–µ—Ç–∞ (–æ–±—ã—á–Ω–æ 14)
