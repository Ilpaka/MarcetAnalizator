# Crypto Trading Bot

Автоматизированный торговый бот для криптовалют с техническим анализом и ML-предсказаниями.

## 🚀 Быстрый старт

### Запуск приложения

1. **Сборка приложения** (если еще не собрано):
   ```bash
   wails build
   ```
   или используйте скрипт:
   ```bash
   build-app.bat
   ```

2. **Запуск приложения**:
   ```bash
   start.bat
   ```
   или напрямую:
   ```bash
   build\bin\crypto-trading-bot.exe
   ```

### Режим разработки

Для разработки с hot-reload:
```bash
wails dev
```

## 📋 Требования

- Go 1.21+
- Node.js 18+
- Wails CLI v2.10+
- Windows 10/11 (для production build)

## 🛠️ Установка зависимостей

### Go зависимости
```bash
go mod download
```

### Frontend зависимости
```bash
cd frontend
npm install
```

## 📁 Структура проекта

```
├── main.go                 # Точка входа приложения
├── app.go                  # Основная логика приложения
├── internal/               # Внутренние пакеты
│   ├── binance/           # Binance API клиент
│   ├── trading/           # Торговый движок
│   ├── strategies/        # Торговые стратегии
│   ├── indicators/        # Технические индикаторы
│   └── ...
├── frontend/              # React frontend
│   └── src/
└── ml_service/            # Python ML сервис
```

## 🎯 Основные возможности

- 📊 Технический анализ с множеством индикаторов
- 🤖 Автономная торговля с настраиваемыми стратегиями
- 📈 Интервальная торговля
- 💹 Paper trading (симуляция торговли)
- 🧠 ML-предсказания цен
- 📱 Современный веб-интерфейс
